# 🚲 Ecobici CDMX Dashboard

A real-time data visualization and analytics dashboard for Mexico City's Public Bicycle Sharing System (Ecobici). Built with Python and Streamlit, this application provides users and city planners with insights into station availability, usage patterns, and system status.

---

## 🚀 Features

* **Real-time Availability:** Live map showing available bikes and free slots at every station across Mexico City.
* **Station Analytics:** Historical usage data, peak hours, and popular routes.
* **Smart Search:** Filter stations by neighborhood (Colonia), availability, or station ID.
* **Responsive Design:** Optimized for both desktop and mobile viewing via Streamlit.
* **Data Integration:** Utilizes the official [Ecobici Open Data API](https://www.ecobici.cdmx.gob.mx/es/informacion-del-servicio/open-data).

## 🛠️ Tech Stack

* **Language:** Python 3.9+
* **Framework:** [Streamlit](https://streamlit.io/)
* **Data Handling:** Pandas, NumPy
* **Visualization:** Plotly, Folium (for interactive maps)
* **API Requests:** Requests
